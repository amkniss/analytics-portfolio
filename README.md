## 🎓 Methodologies

### LTV Calculation
```
LTV = Average Order Value × Purchase Frequency × Customer Lifespan
```

### Customer Segmentation
Based on RFM (Recency, Frequency, Monetary) analysis:
- **R**ecency: Days since last purchase
- **F**requency: Number of orders
- **M**onetary: Total revenue contribution

### Priority Scoring
Weighted algorithm considering:
- Revenue (40%)
- Order frequency (30%)
- LTV potential (30%)
